Nadbooks is a commercial ebook platrom that allows authors to upload books they wrote as well as allow other users to make purchase of these ebooks with either fiat currency or crypto
Users can make upload books they intend to sell.
Books can be purchase with fiat currency via paystack integration as well as `BOOKS` which is a token on monad blockchain

The project is actively being built with using NestJS, TypeScript, and Redis for caching and as well as keeping the auth system stateful. Frontend which I am not exactly part of in terms of development is built wih react
